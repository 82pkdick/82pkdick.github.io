Hexoをやってみた

デプロイでちょいはまる.

hexo deploy -g

で、パーミッション・エラーっぽい

FATAL Something's wrong. Maybe you can find the solution here: http://hexo.io/docs/troubleshooting.html
Error: Warning: Permanently added the RSA host key for IP address '192.30.255.113' to the list of known hosts.
Permission denied (publickey).
fatal: Could not read from remote repository.

Please make sure you have the correct access rights
and the repository exists.

下記サイトを参考にしてエラー回避できた。

Hexo でひさびさに deploy したらパーミッションエラー

[解決方法]

deploy:
  type: git
-  repo: 'git@github.com:hogehoge/fugafuga.github.io.git'
+  repo: 'https://github.com/hogehoge/fugafuga.github.io.git'
  branch: master

要はSSHの代わりにHTTPSのアドレスを使うとうまくいくということらしい。
